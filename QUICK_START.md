# הוראות התחלה מהירה

## התקנה

```bash
npm install
```

## הרצה

```bash
npm run dev
```

פתח את הדפדפן בכתובת: `http://localhost:3000`

## שימוש ראשוני

1. **הוסף שחקנים**: עבור לדף "שחקנים" והוסף שחקנים חדשים
2. **צור אירועים**: עבור לדף "לוח אימונים ומשחקים" והוסף אימונים ומשחקים
3. **סמן נוכחות**: לחץ על "נוכחות" ליד כל אירוע כדי לסמן מי נכח
4. **עדכן סטטיסטיקות**: בדף "סטטיסטיקות" לחץ על מספרים כדי לערוך שערים וכרטיסים

## הערות חשובות

- כל הנתונים נשמרים בדפדפן בלבד (localStorage)
- הנתונים יישמרו גם אחרי סגירת הדפדפן
- אם תמחק את נתוני הדפדפן, הנתונים יאבדו

## פתרון בעיות

אם יש שגיאות:
1. ודא ש-Node.js מותקן (גרסה 18+)
2. מחק את `node_modules` והרץ `npm install` מחדש
3. ודא שכל הקבצים נשמרו כראוי
