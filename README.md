# Haifa FC Manager

אפליקציית ניהול קבוצת מכבי חיפה - כלי ניהול מלא לשחקנים, אימונים, משחקים וסטטיסטיקות.

## תיאור

Haifa FC Manager היא אפליקציית Web מבוססת Next.js + React לניהול קבוצת מכבי חיפה. האפליקציה פועלת לחלוטין בצד הלקוח, ללא צורך בשרת, וכל הנתונים נשמרים מקומית בדפדפן (localStorage).

## תכונות עיקריות

### 🏃 ניהול שחקנים
- הוספה, עריכה ומחיקה של שחקנים
- פרטי שחקן: שם, מספר חולצה, תפקיד, גיל, מצב פציעה
- מיון לפי שם, גיל או מספר חולצה

### 📅 לוח אימונים ומשחקים
- הוספה, עריכה ומחיקה של אירועים (אימונים ומשחקים)
- תצוגת חודש/שבוע
- הדגשת המשחק הבא
- סימון נוכחות לכל שחקן בכל אירוע

### 📊 סטטיסטיקות
- סטטיסטיקות שחקנים: מספר משחקים, שערים, כרטיסים, אחוז נוכחות
- סטטיסטיקות קבוצתיות: מלך השערים, אחוז נוכחות ממוצע, סיכום משחקים
- גרפים ויזואליים עם Recharts
- עריכה ישירה של סטטיסטיקות (שערים, כרטיסים)

### 🎯 דשבורד ראשי
- תצוגת האירוע הבא
- סיכום סטטיסטיקות מהיר
- שחקנים בולטים (לפי שערים ונוכחות)

## טכנולוגיות

- **Next.js 14** - Framework React
- **React 18** - ספריית UI
- **TypeScript** - טייפ-ספייפי
- **Tailwind CSS** - עיצוב
- **Recharts** - גרפים וסטטיסטיקות
- **date-fns** - ניהול תאריכים
- **localStorage** - אחסון מקומי

## התקנה והרצה

### דרישות מוקדמות
- Node.js 18+ 
- npm או yarn

### התקנה

```bash
# התקנת תלויות
npm install

# הרצת שרת פיתוח
npm run dev
```

האפליקציה תרוץ על `http://localhost:3000`

### בנייה לייצור

```bash
npm run build
npm start
```

## שימוש

### הוספת שחקן
1. עבור לדף "שחקנים"
2. לחץ על "הוסף שחקן"
3. מלא את הפרטים ולחץ "הוסף"

### יצירת אירוע
1. עבור לדף "לוח אימונים ומשחקים"
2. לחץ על "הוסף אירוע"
3. בחר תאריך, שעה, מיקום וסוג (אימון/משחק)
4. לחץ "הוסף"

### סימון נוכחות
1. בדף "לוח אימונים ומשחקים"
2. לחץ על "נוכחות" ליד האירוע הרצוי
3. לחץ על כל שחקן כדי לסמן נוכחות/היעדרות

### עדכון סטטיסטיקות
1. עבור לדף "סטטיסטיקות"
2. לחץ על מספר השערים או הכרטיסים כדי לערוך
3. הזן את הערך החדש ולחץ ✓
