# הוראות להוספת הלוגו הרשמי של מכבי חיפה

## אפשרות 1: שימוש בלוגו הקיים (SVG)
הלוגו הנוכחי הוא SVG שנוצר בהתבסס על הלוגו הרשמי של מכבי חיפה.

## אפשרות 2: הוספת הלוגו הרשמי מהאתר

1. היכנס לאתר הרשמי: https://www.mhaifafc.com/
2. הורד את הלוגו הרשמי (PNG או SVG)
3. שמור את הקובץ בשם `maccabi-haifa-logo.png` או `maccabi-haifa-logo.svg`
4. העתק את הקובץ לתיקיית `public/`
5. הקומפוננטה `components/Logo.tsx` תשתמש בלוגו החדש אוטומטית

## הערה
אם תרצה להשתמש בקובץ בשם אחר, עדכן את הנתיב בקובץ `components/Logo.tsx` בשורה:
```tsx
src="/maccabi-haifa-logo.svg"
```
